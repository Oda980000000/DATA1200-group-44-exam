Accessibility Statement

ODGAJE is committed to providing a website that is accessible to
the widest possible audience, regardless of technology or abilities.
We will continiuosly improve our site by acquiring knowledge about
new technologies and gain experience by including these in our scope.
During this project we will strive to provide an
accessible, optimized and well-rounded user experience.

Our Journey towards an Accessible Website

We started by researching the subject matter and found
some key points from the Web Content Accessibility Guidelines (WCAG)
that we'd like to highlight on this page as seen in the box to your right.
Because we started said research some time after initializing the website, some changes were due.

We decided to direct our initial focus on a straightforward menu with logically arranged clickable elements.
While testing out colorschemes we found a theme that could be aestethically pleasing as well as accessible.
We also wanted to make sure the website offered alternative
navigation methods using keyboard shortcuts.
We tested these keyboard features by ensuring that TAB, SPACE and arrow keys could be used to navigate the site and ENTER activated the current selection.
